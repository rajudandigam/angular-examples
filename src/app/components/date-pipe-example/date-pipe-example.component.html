Welcome to Angular Date pipe examples

Current date in Default format

{{ currentDate}}

Current date with date pipe

{{ currentDate | date }}

Current date with date pipe and short format

{{ currentDate | date: 'short'}}

Current date with date pipe and mediumTime format

{{ currentDate | date: 'mediumTime'}}

Current date with date pipe and custom format

{{ currentDate | date: 'yyyy-MM-dd HH:mm:ss'}}

Current date with date pipe and medium format and time zone

{{ currentDate | date: 'medium':'IST' }}

Current date with date pipe and medium format and time zone and locale

{{ currentDate | date: 'medium':'GMT':'en-GB' }}
